# 经典命例高置信 gold cases

当前收录两个彼此独立的高置信集：

- `qiongtong-high-confidence.json`：18 个《图解穷通宝鉴》原局命例。
- `ziping-high-confidence.json`：6 个《图解子平真诠》格局取用命例。

每例均已直接查看 PDF 扫描页，确认四柱与断语属于同一编号或示例块。

这是独立的待接入 fixture，暂不直接并入 `eval/baziprofile-accuracy/gold-cases.js`：先跑独立基线，再决定是进训练集还是冻结集，可避免新样本污染现有指标。

## 可靠性约束

- 证据以 PDF 版面为准，OCR 只用于搜索候选页。
- 字段只保留原文明示的结论；不明示的旺衰、喜忌不猜测。
- `scoringScope` 是可硬评分维度，不在其中的字段只作背景。
- PDF SHA-256 已固化；更换扫描版本后页码必须重新核对。
- 《图解穷通宝鉴》是现代图解编注本，本集的 `theory_profile` 不等同于无注古本。

## 未收录候选

- 扫描图与 OCR 文本层错位的页面，即使 OCR 能连出四柱也不收录。
- 四柱与解读分属不同版面块、无法确证对应关系的命例。
- 现代名人图解页：除了转述还可能涉及生时口径，本批不用。
- 只有富贵结论、未给出可编码格局/用神依据的命例。
